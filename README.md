# JusticeCompanion

## Requirements
Check `requirements.txt`

## Setup

Make a virtual environment:
```sh
python -m venv .venv/
```
Install required modules and stubs:
```sh
./.venv/Scripts/python -m pip install -r ./requirements.txt
```
